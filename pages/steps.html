じゃあ、実際の手順について

Settings を押下する
  • {RepositoryURL}/settings へ移動します
Pages を押下する
  • {RepositoryURL}/settings/pages へ移動します
Branch / None ▼ を押下する
  • {RepositoryURL}/settings/pages へ移動します
対象ブランチを選択
  • 複数で開発する場合などは Branch は複数になりますが、この資料では main 以外は未作成のため、main Branchを選択します。
Save ボタンを押下する
GitHub Pages source saved が表示されれば公開成功
  • ※実際に公開されるには多少待つ必要があります(待ったうえでリロード等を行ってください)
公開されたサイトを見に行く
  • リンクをクリックして公開されたサイトを確認します
