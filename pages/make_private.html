非公開にする

  • Settings を押下する
    • {RepositoryURL}/settings へ移動
  • Danger Zone → Change repository visibility
    • "Change visibility" を押下
    • "Change to private" ボタンを押下する
    • 確認ダイアログ(Make <<repository name>> private)の手順:
      • "I want to make this repository private" を選択
      • "I have read and understand these effects" を確認して選択
      • "Make this repository private" を押下
    • 最初のページへ戻って、リポジトリ名の横に鍵のマークがついていれば完了
