非公開にする

  • Settings を押下する

    • {RepositoryURL}/settings へ移動します

  • Danger Zone → Change repository visibility を押下する

  • Change visibility を押下する

    • Change to private (コンテキストメニュー)が表示されます

  • "Change to private" ボタンを押下する

  • "I want to make this repository private" を選択

    • ポップアップが表示されます

  • "I have read and understand these effects" を確認して選択

    • ポップアップの内容が変化します

  • "Make this repository private" を押下

    • {RepositoryURL}/settings へ移動します
    • リポジトリ名の横に鍵のマークがついていればOK
